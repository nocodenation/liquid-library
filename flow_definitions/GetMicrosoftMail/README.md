# GetMicrosoftMail

This flow retrieves emails from a Microsoft account using the Microsoft Graph API. It utilizes a previously obtained OAuth token to authenticate the request.

## Processors

-   **GetMicrosoftMail** ![Liquid Library](https://img.shields.io/badge/liquid--library-blue): A custom Python processor that fetches emails from the specified folder.

## Configuration

The `GetMicrosoftMail` processor is configured with the following properties:

-   **Filter Query**: `isRead eq false` (Fetches only unread emails)
-   **Token File Path**: `/files/MicrosoftOAuthManager/microsoft-token.json` (Path to the OAuth token file)
-   **Output Format**: `JSON`
-   **Folder ID**: `Inbox`
-   **Mark as Read**: `false`

## Dependencies

This flow depends on a valid OAuth token file being present at the specified `Token File Path`. This token is typically generated by the **Microsoft_OAuth** flow.
